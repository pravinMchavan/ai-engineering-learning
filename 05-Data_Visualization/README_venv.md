# Data Visualization Virtual Environment

## Setup Instructions

This virtual environment contains all the packages needed for data visualization work.

## How to Activate

### Windows PowerShell:
```powershell
cd c:\Pravin\Course\ai-engineering-learning
.\.venv-pv\Scripts\Activate.ps1
```

### Command Prompt:
```cmd
cd c:\Pravin\Course\ai-engineering-learning
.venv-pv\Scripts\activate.bat
```

## Installed Packages
- numpy: Numerical computing
- pandas: Data manipulation and analysis
- matplotlib: Basic plotting
- seaborn: Statistical visualization
- plotly: Interactive visualizations
- jupyter: Interactive notebooks
- scikit-learn: Machine learning
- scipy: Scientific computing

## Usage
After activation, you can run Python scripts or Jupyter notebooks with all packages available.

## Deactivate
To exit the virtual environment:
```powershell
deactivate
```
