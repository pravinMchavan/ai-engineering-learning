1. WHAT IS A VECTOR?
A vector is an ordered collection of numbers.
It can represent direction, position, features, or model weights.

Example:
v = [2, 3]

Text diagram:

      y
      ^
      |
  3   |        * (2, 3)
      |      /
      |    /
      |  /
      |/
------O--------------------> x

O is the origin (0, 0)


2. TYPES OF VECTORS
There are many types of vectors. These are the basic ones:

1. Zero vector
All values are zero.
Example: [0, 0] or [0, 0, 0]

2. Unit vector
A vector whose length is 1.
Example: [1, 0]

3. Row vector
Written in one row.
Example: [1, 2, 3]

4. Column vector
Written in one column.
Example:
[[1],
 [2],
 [3]]

5. Equal vectors
Two vectors are equal if they have the same values, same magnitude, and same direction.
Example:
[2, 3] and [2, 3]

6. Negative vectors
Two vectors are negative to each other if they have the same magnitude but opposite direction.
Example:
[2, 3] and [-2, -3]

Text diagram:
Zero vector       = [0, 0]
Unit vector       = length 1
Row vector        = [1, 2, 3]
Column vector     = [[1], [2], [3]]
Equal vectors     = same values and same direction
Negative vectors  = opposite direction


3. ROW VECTOR, COLUMN VECTOR, AND MATRIX
These three ideas are closely connected.

Row vector:
A vector written horizontally.
Example:
[1  2  3]
It has 1 row and 3 columns.

Column vector:
A vector written vertically.
Example:
[1]
[2]
[3]
It has 3 rows and 1 column.

Matrix:
A rectangular arrangement of numbers in rows and columns.
Example:
[1  2  3]
[4  5  6]
It has 2 rows and 3 columns.

Important relation:
A vector is a special kind of matrix.
Row vector = 1 x n matrix
Column vector = n x 1 matrix

Easy memory trick:
Row = horizontal
Column = vertical
Matrix = full table of numbers


4. SPAN OF VECTORS
Span means all possible vectors we can make by using scalar multiplication and addition.

If we have vectors v1 and v2, then their span is all vectors of the form:
a*v1 + b*v2

Example 1:
v1 = [1, 0]
v2 = [0, 1]

Then:
a*v1 + b*v2
= a*[1, 0] + b*[0, 1]
= [a, b]

This means we can reach any point in the 2D plane.
So the span of [1, 0] and [0, 1] is the whole 2D plane.

Example 2:
If we only have [1, 0], then we can only move left or right on the x-axis.
So the span is only a line.

Text diagram:
Using one vector:
[1, 0] -> line

Using two independent vectors:
[1, 0] and [0, 1] -> full plane

Easy idea:
Span = all places we can reach using the given vectors.


5. DIMENSION OF A VECTOR
Dimension means the number of values inside the vector.

[5] -> dimension 1
[2, 3] -> dimension 2
[2, 3, 4] -> dimension 3

Text diagram:
[5]           -> 1D
[2, 3]        -> 2D
[2, 3, 4]     -> 3D
[1, 5, 9, 2]  -> 4D


6. VECTOR ADDITION
Add the matching positions.

Example:
[2, 3] + [4, 1] = [6, 4]

Text diagram:
[2, 3]
[4, 1]
------
[6, 4]

Geometric meaning:
Place the second vector at the tip of the first vector.
The result from the origin to the final tip is the sum vector.


7. VECTOR SUBTRACTION
Subtract the matching positions.

Example:
[2, 3] - [4, 1] = [-2, 2]

Text diagram:
[2, 3]
[4, 1]
------
[-2, 2]

Meaning:
Subtraction tells us the change needed to go from one vector to another.


8. SCALAR MULTIPLICATION
A scalar is just one number.
Multiply every value of the vector by that number.

Example:
2 * [2, 3] = [4, 6]

Text diagram:
v  = [2, 3]
2v = [4, 6]

Same direction, larger length


9. DOT PRODUCT
Multiply matching values and add them.

Example:
[2, 3] . [4, 1]
= (2 x 4) + (3 x 1)
= 8 + 3
= 11

Text diagram:
First multiply:
2 x 4 = 8
3 x 1 = 3

Then add:
8 + 3 = 11

Why it matters:
The dot product is used in similarity, linear models, and neural networks.


10. WHY VECTORS MATTER IN ML
Vectors can store:
- one data point
- one image in number form
- one word embedding
- one model weight list

Example:
[age, study_hours, attendance]
[19, 6, 92]

This is one data point written as a vector.
