1. WHAT IS A DERIVATIVE?
A derivative measures how a function changes as its input changes.
It represents the instantaneous rate of change or slope of the tangent line.

For a function f(x), the derivative is denoted as f'(x) or df/dx.

Example:
f(x) = x^2
f'(x) = 2x

At x = 3, f'(3) = 6, meaning the slope is 6.

2. HOW TO CALCULATE DERIVATIVES
Basic rules:

1. Power rule: d/dx(x^n) = n*x^(n-1)
Example: d/dx(x^2) = 2x

2. Constant rule: d/dx(c) = 0
Example: d/dx(5) = 0

3. Sum rule: d/dx(f + g) = f' + g'
Example: d/dx(x^2 + 3x) = 2x + 3

4. Product rule: d/dx(f*g) = f'*g + f*g'
Example: d/dx(x*sin(x)) = 1*sin(x) + x*cos(x)

5. Quotient rule: d/dx(f/g) = (f'*g - f*g') / g^2
Example: d/dx(x^2/x) = (2x*x - x^2*1)/x^2 = x^2/x^2 = 1


3. PARTIAL DERIVATIVES
For functions of multiple variables, partial derivatives measure change with respect to one variable while keeping others constant.

For f(x,y) = x^2 + y^3

Example:

‚ąāf/‚ąāx = 2x*y + 3

4. INTRODUCTION TO GRADIENT
The gradient is a vector of all partial derivatives.
It points in the direction of steepest ascent.

For f(x,y), gradient ‚ąáf = (‚ąāf/‚ąāx, ‚ąāf/‚ąāy)

Example:
f(x,y) = x^2 + y^2
‚ąáf = (2x, 2y)

At (1,1), gradient = (2,2), pointing away from origin.

5. CHAIN RULE OF DERIVATIVES
The chain rule allows differentiation of composite functions.

If y = f(u) and u = g(x), then dy/dx = dy/du * du/dx

Example:
y = (x^2 + 1)^3
Let u = x^2 + 1, y = u^3
dy/dx = dy/du * du/dx = 3u^2 * 2x = 3(x^2 + 1)^2 * 2x

For multiple variables:
If z = f(x,y) and x = g(t), y = h(t)
dz/dt = ‚ąāz/‚ąāx * dx/dt + ‚ąāz/‚ąāy * dy/dt
