=====================================================
HYPOTHESIS TESTING - COMPLETE GUIDE
=====================================================

1. DEFINITION
=====================================================

Hypothesis Testing is a statistical method used to make decisions about a population 
based on sample data. It involves formulating two competing hypotheses and determining 
which one is more likely to be true based on evidence from sample data.

Key Components:

a) Null Hypothesis (H₀)
   - A statement that there is no effect, no difference, or no relationship
   - Always assumes the status quo or no change
   - Example: "The average height of males is equal to the average height of females"

b) Alternative Hypothesis (H₁ or Hₐ)
   - A statement that contradicts the null hypothesis
   - Claims there IS an effect, difference, or relationship
   - Example: "The average height of males is NOT equal to the average height of females"

c) Significance Level (α)
   - The probability of rejecting the null hypothesis when it's actually true (Type I Error)
   - Common values: 0.05 (5%), 0.01 (1%), 0.10 (10%)

d) P-value
   - The probability of observing sample data as extreme as what we observed, 
     assuming the null hypothesis is true
   - If p-value < α, we reject the null hypothesis
   - If p-value ≥ α, we fail to reject the null hypothesis

e) Test Statistic
   - A standardized value calculated from sample data
   - Examples: t-statistic, z-statistic, F-statistic, chi-square statistic

f) Critical Value
   - Threshold value that defines the rejection region
   - Depends on the significance level and type of test


2. TYPES OF ERRORS
=====================================================

Type I Error (False Positive):
- Rejecting a true null hypothesis
- Probability = α (significance level)
- Saying something is significantly different when it's actually not

Type II Error (False Negative):
- Failing to reject a false null hypothesis
- Probability = β
- Missing a true effect or difference

Power of Test = 1 - β
- Probability of correctly rejecting a false null hypothesis
- Researchers typically aim for power = 0.80 (80%)


3. STEPS IN HYPOTHESIS TESTING
=====================================================

Step 1: State the Hypotheses
   - Write H₀ and H₁ clearly

Step 2: Choose Significance Level
   - Typically α = 0.05

Step 3: Select Appropriate Test
   - Based on data type, sample size, and distribution

Step 4: Calculate Test Statistic
   - Using sample data

Step 5: Determine P-value
   - Using test statistic and probability distribution

Step 6: Make a Decision
   - If p-value < α: Reject H₀ (statistically significant)
   - If p-value ≥ α: Fail to reject H₀ (not statistically significant)

Step 7: Draw Conclusion
   - Interpret results in context of the problem


4. COMMON HYPOTHESIS TESTS
=====================================================

a) One-Sample t-test
   - Purpose: Test if a sample mean differs from a known population mean
   - Formula: t = (x̄ - μ) / (s / √n)
   - Where: x̄ = sample mean, μ = population mean, s = sample std dev, n = sample size

b) Two-Sample t-test
   - Purpose: Compare means of two independent samples
   - Used when: Testing if two groups have different means

c) Paired t-test
   - Purpose: Compare means of two related samples
   - Used when: Same subjects measured at two different times

d) Chi-Square Test
   - Purpose: Test relationship between categorical variables
   - Used when: Working with frequency data

e) ANOVA (Analysis of Variance)
   - Purpose: Compare means of three or more groups
   - Used when: Testing multiple group means simultaneously

f) Z-test
   - Purpose: Test sample mean when population std dev is known
   - Used when: Large samples (n > 30) or known population parameters

g) Correlation Test
   - Purpose: Test if correlation between two variables is significant
   - Provides p-value for correlation coefficient


5. CONFIDENCE INTERVAL vs HYPOTHESIS TEST
=====================================================

Confidence Interval:
- Provides a range of values likely to contain the true parameter
- Example: "We are 95% confident the true mean is between 50 and 60"
- Related to β error

Hypothesis Test:
- Tests a specific claim about a parameter
- Provides yes/no answer based on p-value
- Related to α error

Relationship: If 95% CI doesn't include hypothesized value, we reject H₀ at α = 0.05


6. EXAMPLES WITH CALCULATIONS
=====================================================

EXAMPLE 1: One-Sample t-test
─────────────────────────────

Scenario:
A company claims their light bulbs last 1000 hours on average.
We test a sample of 25 bulbs and find:
- Sample mean (x̄) = 985 hours
- Sample std dev (s) = 50 hours
- Significance level = 0.05

H₀: μ = 1000 hours (company's claim is true)
H₁: μ ≠ 1000 hours (company's claim is false)

Calculation:
t = (985 - 1000) / (50 / √25)
t = -15 / (50 / 5)
t = -15 / 10
t = -1.5

Degrees of freedom = n - 1 = 24
Critical value for two-tailed test at α = 0.05: ±2.064

Decision: Since -1.5 > -2.064, we fail to reject H₀
Conclusion: There is insufficient evidence to reject the company's claim.


EXAMPLE 2: Two-Sample t-test
────────────────────────────

Scenario:
Compare exam scores between two teaching methods
Group A (Traditional): n₁ = 30, x̄₁ = 78, s₁ = 8
Group B (Online): n₂ = 30, x̄₂ = 75, s₂ = 10

H₀: μ₁ = μ₂ (no difference between teaching methods)
H₁: μ₁ ≠ μ₂ (there is a difference)

t = (x̄₁ - x̄₂) / √[(s₁²/n₁) + (s₂²/n₂)]
t = (78 - 75) / √[(64/30) + (100/30)]
t = 3 / √[2.133 + 3.333]
t = 3 / √5.467
t ≈ 1.28

If p-value = 0.15 > 0.05, we fail to reject H₀
Conclusion: There is no statistically significant difference between teaching methods.


EXAMPLE 3: Chi-Square Test
──────────────────────────

Scenario:
Testing if customer preference for product colors is independent of gender

                Red    Blue   Green   Total
Male            50     40      35     125
Female          45     55      45     145
Total           95     95      80     270

H₀: Color preference is independent of gender
H₁: Color preference depends on gender

Chi-square = Σ[(Observed - Expected)² / Expected]

If χ² = 4.85 and critical value = 5.99 (α = 0.05, df = 2)
Then χ² < critical value, so we fail to reject H₀
Conclusion: Color preference appears to be independent of gender.


7. REAL-WORLD STUDY CASES
=====================================================

CASE STUDY 1: Medical Drug Effectiveness
──────────────────────────────────────────

Background:
A pharmaceutical company develops a new cholesterol-reducing drug.
FDA requires evidence that it reduces cholesterol by >15 mg/dL on average.

Method:
- Sample: 100 patients with high cholesterol
- Before treatment: Mean cholesterol = 240 mg/dL (std dev = 30)
- After treatment: Mean cholesterol = 215 mg/dL (std dev = 25)
- Paired t-test with α = 0.05

Hypotheses:
H₀: μ_difference ≤ 15 mg/dL (drug isn't effective enough)
H₁: μ_difference > 15 mg/dL (drug is effective)

Results:
- Actual difference = 25 mg/dL
- t-statistic = 3.33
- p-value < 0.001

Decision: Reject H₀
Conclusion: The drug significantly reduces cholesterol by more than 15 mg/dL.
FDA Approval: GRANTED (with strong statistical evidence)


CASE STUDY 2: A/B Testing in E-commerce
────────────────────────────────────────

Background:
An online retailer wants to test if a new website design increases sales.

Method:
- Control Group (Old Design): 5000 visitors, conversion rate = 3.2%
- Test Group (New Design): 5000 visitors, conversion rate = 3.8%
- Two-proportion z-test with α = 0.05

Research Question:
Does the new design increase conversion rate?

Hypotheses:
H₀: p₁ = p₂ (new design has same conversion rate)
H₁: p₁ ≠ p₂ (new design has different conversion rate)

Results:
- Control conversions = 160, Test conversions = 190
- z-statistic = 2.14
- p-value = 0.032

Decision: Reject H₀ (p-value < 0.05)
Conclusion: The new design significantly increases conversion rate.
Business Action: Roll out new design to all website traffic


CASE STUDY 3: Crop Yield Improvement
─────────────────────────────────────

Background:
Agricultural company tests if new fertilizer increases crop yield.

Method:
- Sample: 40 fields
- Traditional fertilizer: mean yield = 45 tons/acre, std dev = 6 tons
- New fertilizer: mean yield = 49 tons/acre, std dev = 7 tons
- Two-sample t-test with α = 0.05

Hypotheses:
H₀: μ_new = μ_traditional (fertilizers produce equal yields)
H₁: μ_new > μ_traditional (new fertilizer is better)

Results:
- t-statistic = 2.45
- p-value = 0.008

Decision: Reject H₀
Conclusion: New fertilizer produces significantly higher yields.
Economic Impact: Potential profit increase of $180/acre if adopted widely


CASE STUDY 4: Quality Control (Chi-Square)
───────────────────────────────────────────

Background:
Manufacturing plant checks if product defects are equally distributed across shifts.

Method:
- Inspected 500 products
- Morning shift: 45 defects
- Afternoon shift: 55 defects
- Night shift: 50 defects
- Chi-square goodness of fit test with α = 0.05

Hypotheses:
H₀: Defect rate is equal across all shifts
H₁: Defect rate differs among shifts

Expected defects per shift = 500/3 ≈ 166.67
Each shift should have ~50 defects

Results:
- χ² = 1.25
- Critical value = 5.99 (df = 2)
- p-value = 0.535

Decision: Fail to reject H₀
Conclusion: No significant difference in defect rates across shifts.
Management Decision: Current shift scheduling and processes are adequate


CASE STUDY 5: Marketing Campaign Effectiveness
───────────────────────────────────────────────

Background:
Company runs targeted email campaign and tracks customer engagement.

Method:
- Existing customers: 60 opened email, 150 not opened (40% open rate)
- After optimization: 120 opened, 150 not opened (44.4% open rate)
- Two-proportion test with α = 0.05

Sample sizes: n₁ = 210, n₂ = 270

Hypotheses:
H₀: p₁ = p₂ (optimization doesn't improve open rate)
H₁: p₁ < p₂ (optimization improves open rate)

Results:
- z-statistic = 1.42
- p-value = 0.078

Decision: Fail to reject H₀ (just above significance threshold)
Conclusion: Not enough evidence that optimization improves open rates.
Business Action: Continue tested campaign but refine further; consider larger sample


8. COMMON MISTAKES IN HYPOTHESIS TESTING
=====================================================

1. P-hacking: Running multiple tests and only reporting significant ones
2. Misinterpreting p-value: p-value is NOT probability H₀ is true
3. Ignoring effect size: Statistical significance ≠ practical significance
4. Fixed sample size: Always determine sample size before collecting data
5. Publication bias: Only significant results get published
6. Multiple comparisons: Adjust significance level when running multiple tests
7. Confusing correlation with causation: Hypothesis tests show associations, not causation
8. Assuming normality: Many tests require approximately normal distribution
9. Ignoring assumptions: Each test has specific assumptions that must be checked
10. Failing to consider context: Always interpret results within real-world context


9. KEY TAKEAWAYS
=====================================================

✓ Hypothesis testing is about making evidence-based decisions
✓ Choose appropriate test based on data type and research question
✓ Always state hypotheses BEFORE looking at data
✓ P-value tells us strength of evidence against null hypothesis
✓ Statistical significance ≠ practical significance
✓ Report both p-value AND effect size
✓ Consider both Type I and Type II errors
✓ Use confidence intervals alongside hypothesis tests
✓ Real-world applications require careful interpretation
✓ Document all decisions and assumptions clearly
