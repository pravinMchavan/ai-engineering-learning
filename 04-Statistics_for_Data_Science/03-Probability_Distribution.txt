Random Variable

Definition:
A random variable is a function that assigns a numerical value to each possible outcome of a random experiment.

Explanation:
- A random variable converts outcomes from a random process into numbers.
- It is not a variable that we choose; it is defined by the process and its possible outcomes.
- There are two main types of random variables:
  * Discrete random variable: has countable outcomes (like 0, 1, 2, ...).
  * Continuous random variable: has infinitely many values in a range.

Examples:
- Rolling a fair six-sided die: the random variable X = outcome {1, 2, 3, 4, 5, 6}.
- Counting the number of heads in 3 coin flips: Y = 0, 1, 2, or 3.
- Measuring the height of a randomly selected student: Z can be any value in a range.

Probability

Definition:
Probability is the measure of how likely an event is to happen. It is a number between 0 and 1, where 0 means impossible and 1 means certain.

Explanation:
- Probability describes the chance of an event in a random experiment.
- For equally likely outcomes, probability = (number of favorable outcomes) / (total number of outcomes).
- The sum of probabilities of all possible outcomes of a random variable is 1.

Formula:
P(E) = number of favorable outcomes / total number of possible outcomes

Solved Example 1:
Problem: What is the probability of rolling a 4 on a fair six-sided die?
Solution: favorable outcome = {4}, total outcomes = {1,2,3,4,5,6}
P(rolling 4) = 1 / 6 � 0.1667

Solved Example 2:
Problem: What is the probability of drawing an Ace from a standard 52-card deck?
Solution: favorable outcomes = 4 Aces, total outcomes = 52 cards
P(drawing an Ace) = 4 / 52 = 1 / 13 � 0.0769

- 0 = P(E) = 1
- P(Sample Space) = 1
- P(Not E) = 1 - P(E)

Relationship to Random Variables:
- A random variable has a probability distribution that gives probabilities for each possible value.
- For a discrete random variable X, P(X = x) is the probability that X takes the value x.
Probability Distribution Types

1) Discrete Probability Distribution
Definition:
A discrete probability distribution describes the probabilities of outcomes for a discrete random variable.

Function:
- Probability Mass Function (PMF): gives the probability that a discrete random variable equals a specific value.
- Notation: P(X = x) = f(x)

Properties:
- f(x) = 0 for all x
- The sum of f(x) over all possible values equals 1

Example:
For a fair six-sided die, X = outcome, P(X = x) = 1/6 for x = 1,2,3,4,5,6.

2) Continuous Probability Distribution
Definition:
A continuous probability distribution describes the probabilities of outcomes for a continuous random variable.

Function:
- Probability Density Function (PDF): describes the relative likelihood of a continuous random variable taking a value.
- Notation: f(x) and probability over an interval is P(a = X = b) = ?_a^b f(x) dx

Properties:
- f(x) = 0 for all x
- The total area under f(x) over all values equals 1
- For a specific value x, P(X = x) = 0; probabilities are defined for intervals instead.

Example:
If heights are modeled by a continuous distribution, the PDF gives the relative likelihood of each height range.

3) Common Discrete Distributions
- Binomial distribution: number of successes in n independent trials.
- Poisson distribution: count of events in a fixed interval when events occur at a constant average rate.

4) Common Continuous Distributions
- Uniform distribution: every value in a range is equally likely.
- Normal distribution: bell-shaped distribution defined by mean and standard deviation.
